Detecting and Explaining Anomalies Caused by Web Tamper Attacks via Building Consistency-based Normality
Yifan Liao, Ming Xu, Yun Lin, Xiwen Teoh, Xiaofei Xie, Ruitao Feng, Hongyu Zhang, Jinsong Dong
ASE'24 (CCF-A)
Project Page | Paper | Code

This project aims to detect and explain attack-induced anomalies in web applications by learning normal behavioral during runtime. We model the normal constraints with first-order logic, and generate executable Python scripts. By leveraging LLMs in the learning process, this approach enhances both the detection and explanation of anomalies.

Towards Stealthy and Effective Backdoor Attacks on Lane Detection: A Naturalistic Data Poisoning Approach
Yifan Liao, Yuxin Cao, Yedi Zhang, Wentao He, Yan Xiao, Zhiyong Huang, Jinsong Dong
Preprint, 2025
Project Page(TBD) / Paper(TBD)

This project aims to expose and evaluate backdoor vulnerabilities in lane detection (LD) models by generating naturalistic attack triggers through diffusion-based data poisoning. We identify optimal trigger locations using gradient-based heatmaps and synthesize visually plausible triggers via region-aware diffusion editing. By introducing losses that preserve lane structure and scene consistency, the approach ensures stealthy yet effective attacks. This framework enhances both the realism and impact of backdoor attacks, revealing critical weaknesses in state-of-the-art LD systems.
